О проекте

neirostat.ru — это бесплатный онлайн-инструмент для автоматической генерации статей, оптимизированных для публикации на платформе Яндекс.Дзен.

Как это работает?

Сервис использует передовые языковые модели (GPT-4o-mini) через API GitHub Models. Вы указываете тему (или оставляете поле пустым — тогда тема подбирается автоматически из актуальных трендов в области психологии), и искусственный интеллект создаёт полноценную статью.

Возможности

  • 🎯 Генерация по теме — укажите тему, и ИИ напишет развёрнутую статью.
  • 🔀 Случайная тема — оставьте поле пустым, и сервис подберёт тему из трендов.
  • 📋 Редактирование — результат можно отредактировать прямо в браузере.
  • ⬇️ Скачивание — статью можно скачать в формате Markdown (.md).
  • 📋 Копирование — одним кликом скопируйте текст в буфер обмена.

Технологии

  • Backend: Python, Flask
  • AI-модель: GPT-4o-mini (GitHub Models)
  • Аналитика: Яндекс.Метрика
  • Реклама: Яндекс.Реклама (РСЯ)

Для кого этот сервис?

Сервис предназначен для авторов, контент-менеджеров, маркетологов и всех, кто публикует материалы на Яндекс.Дзен и хочет ускорить процесс создания контента.

Открытый код

Проект доступен на GitHub. Вы можете изучить код, предложить улучшения или создать собственную версию.
